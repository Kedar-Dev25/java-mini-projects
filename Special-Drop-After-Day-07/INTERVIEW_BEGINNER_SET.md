## Core Java Questions

Q1. What is JVM?  
Java Virtual Machine executes Java bytecode.

Q2. Difference between int and double?  
int stores integers. double stores decimal values.

Q3. What is an array?  
A collection of same data type elements stored in contiguous memory.

Q4. Why are loops important?  
They allow repeated execution until a condition is satisfied.

Q5. Why is main method static?  
Because it runs without creating an object.

---

## Logic-Based Questions

Q6. Why validate withdrawal in ATM program?  
To prevent invalid transactions.

Q7. Why cast total to double in average calculation?  
To avoid integer division.

Q8. When should we use switch instead of if-else?  
When handling multiple fixed choices.
